# Logs Directory

This directory contains application logs for the Algiers University Attendance System.

## Log Files

- `database.log` - Database connection and query logs
- `auth.log` - Authentication and session logs
- `system.log` - General system activity logs
- `error.log` - Application errors and exceptions

## Log Rotation

Logs are automatically managed by the application. Old logs are kept for audit purposes.

*Note: This directory is ignored by Git for security reasons.*
